How to issue a code

  1. Enter the customer's name or email in the field, then click Generate Code
  2. Click Copy next to the new code and send it to your customer
  3. Click Export for Config to get all codes → paste the accessCodes block into js/fsbo-config.js
  4. Commit the updated config to GitHub — Vercel auto-deploys in ~30 seconds
